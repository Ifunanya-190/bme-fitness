# BME Fitness

A complete fitness website with responsive design and Firebase authentication.

## Features
- Responsive design with mobile menu
- Firebase authentication system
- User profile and progress pages
- Trainer booking system
- Contact forms with guest/user logic
- Theme switching (light/dark)
- All pages styled and functional
- Font Awesome icons integration
- WhatsApp booking integration

## Tech Stack
- HTML5
- CSS3 with custom properties
- JavaScript (ES6+)
- Firebase Authentication
- Responsive design
- No traditional backend (Firebase-only)

## Pages
- Home (index.html)
- About (about.html)

- Contact (contact.html)

## Getting Started
1. Clone this repository
2. Open index.html in your browser
3. Set up Firebase configuration
4. Enjoy the website!
