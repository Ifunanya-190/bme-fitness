# GitHub Push Commands for BME Fitness

## After creating your GitHub repository, run these commands:

### Add GitHub as remote (replace YOUR_USERNAME with your GitHub username):
```bash
git remote add origin https://github.com/YOUR_USERNAME/bme-fitness.git
```

### Push to GitHub:
```bash
git branch -M main
git push -u origin main
```

## Alternative: If you want to use GitHub CLI:
```bash
gh repo create bme-fitness
git remote add origin https://github.com/YOUR_USERNAME/bme-fitness.git
git push -u origin main
```
